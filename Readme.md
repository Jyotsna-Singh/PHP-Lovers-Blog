# Secret Diary

Secret Diary is an online diary application built with PHP. Users can register to create an account and login to store their thoughts in the diary.

* Features/Technologies: 
  * PHP Syntax, PHPMyAdmin, MySQL
  * HTML/CSS
  * Bootstrap
  * Custom Font
